    Class InstructionBundle

    Represents a single interactive visual block of code within the UI.

    Inheritance
    object
    Object
    Component
    Behaviour
    MonoBehaviour
    InstructionBundle
    Namespace: LogicForge.Bundles
    Assembly: LogicForge.dll
    Syntax
    [RequireComponent(typeof(CanvasGroup))]
[RequireComponent(typeof(BaseInstruction))]
public class InstructionBundle : MonoBehaviour
    Remarks

    This component bridges the visual UI representation (Color, Drag and Drop, Layout) with the underlying logical data (BaseInstruction). It implements Unity's Event Interfaces to handle complex drag-and-reorder interactions.
