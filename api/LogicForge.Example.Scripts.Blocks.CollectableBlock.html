    Remarks

    This block bridges the gameplay grid and the variable system. It uses a BaseVariableWatcher to determine if the item has already been collected (persisted state) and a VariableSetter to update the state when the player enters the tile.

    Ideal for implementation of keys, currency, or one-time quest triggers.
