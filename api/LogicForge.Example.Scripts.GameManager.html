    Class GameManager

    The central singleton manager for the level execution context.

    public class GameManager : MonoBehaviour
    Remarks

    The GameManager class acts as the "God Object" for the current puzzle level. Its responsibilities include:

    • Maintaining a registry of all blocks via a coordinate dictionary.
    • Managing the global game state (Win/Fail conditions).
    • Interfacing with the CodeRunner to reset the board when code execution restarts.
