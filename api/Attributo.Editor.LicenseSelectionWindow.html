    Class LicenseSelectionWindow

    An interactive wizard used to assign licenses to project assets during or after import.

    Remarks

    This window serves as the primary user interface for the LicenseImportDetector. It allows developers to:

    • Search and select from a project-wide database of LicenseData.
    • Quickly generate new license templates based on common SPDX standards.
    • Exempt specific folders from future scans using the 'Mute' feature.
