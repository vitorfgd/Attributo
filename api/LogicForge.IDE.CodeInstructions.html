    Class CodeInstructions

    Manages a specific collection of visual instruction blocks (e.g., the "Main" block or a "Loop" body).

    Inheritance
    object
    Object
    Component
    Behaviour

    Syntax
    public class CodeInstructions : MonoBehaviour
    Remarks

    This component synchronizes the visual hierarchy (Unity UI transforms) with the logical data list (InstructionBundles). It handles the insertion of new instructions, placeholder visualization during drag-and-drop, and order updates.
