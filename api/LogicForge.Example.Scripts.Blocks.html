    Namespace LogicForge.Example.Scripts.Blocks

    Classes

    BaseBlock

    Serves as the abstract base class for all grid-based entities within the level.

    CollectableBlock

    Represents a consumable item within the grid that modifies global game state upon collection.

    EnemyBlock

    Represents a hazardous tile that causes an immediate fail state upon player contact.

    FinishBlock

    Represents the objective point of the stage. Reaching this block concludes the level.

    SpawnBlock

    Marker block designating the player's starting position within the grid.

    TeleportationBlock

    A functional block that instantly relocates the player to a specified grid coordinate.

    UnreachableBlock

    Represents a non-traversable structural element, such as walls, pillars, or pits.

    UsableBlock

    Represents an interactive object (e.g., lever, button, switch) that requires manual activation.

    VariableGateBlock

    A conditional structural block that toggles its traversable state based on a global variable.

    WalkableBlock

    Represents a standard floor tile where the player can move freely.
