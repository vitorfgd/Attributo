    Struct VariableValue

    A serializable wrapper acting as a discriminated union for Unity-supported primitive types.

    Namespace: LogicForge.Variables
    Assembly: LogicForge.dll
    Syntax
    [Serializable]
public struct VariableValue
    Remarks

    Unity's serializer does not support polymorphism or 'object' fields natively without complex workarounds. This struct exposes explicit fields for all supported types, using the Type field to determine which field is currently active/valid.
