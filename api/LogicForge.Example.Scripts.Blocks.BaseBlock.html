    Class BaseBlock

    Serves as the abstract base class for all grid-based entities within the level.

    Syntax
    [DisallowMultipleComponent]
public class BaseBlock : MonoBehaviour
    Remarks

    This class handles the fundamental responsibilities of a block, including:

    • Storing grid coordinates.
    • Managing lifecycle events (OnEnter, OnExit).
    • Providing dependency injection for the GameManager controller.

    Inherit from this class to create custom tiles such as traps, items, or structural elements.
