    Attributo: Pro License Manager

    Attributo is an Editor-only Unity tool that automates asset attribution and third-party license compliance. By linking folders to license data and tracking imports, it helps keep your project audit-ready and simplifies generating attribution documents (e.g. ThirdPartyNotices.txt or CREDITS.md).

    Features

    • Import-time prompts – When you add assets under an unlicensed folder, a wizard appears so you can assign or create licenses.
    • Hierarchical licensing – Place a License Link in a folder; that folder and its children inherit that license until overridden by another link deeper in the tree.
    • SPDX integration – Fetch official license text from SPDX by entering an identifier (e.g. MIT, Apache-2.0) in the License Data inspector.
    • License Hub – Central window to audit all links and export a single attribution file with deduplicated license text and notes.
    • Mute folders – Optionally mark a folder as “no license needed” so the import detector does not prompt again for that path.

    Technical note

    Attributo is Editor-only. The Runtime assembly (Attributo) contains only data types (LicenseData, LicenseLink). All hierarchy resolution and Asset Database usage live in the Editor assembly (Attributo.Editor), so the package has no impact on player builds or build size.

    Legal disclaimer

    Attributo is an organizational aid and does not constitute legal advice. You are responsible for verifying that all assets are correctly attributed and that license terms are followed. Always review exported attribution files before shipping.
